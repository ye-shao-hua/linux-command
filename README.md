## 基础指令

### 文件和目录管理
|指令|作用|
|:-:|:-:|
|pwd|显示当前路径|
|cd|切换目录|
|[ls](#ls)|显示当前文件夹下所有文件|
|[mkdir](#mkdir)|创建文件夹|
|[cp](#cp)|复制文件|
|mv|移动文件|
|[rm](#rm)|删除文件|
|touch|创建文件|
|[find](#find)|查找文件|


### a
|whoami|显示当前用户|

## 使用示例

### 基础指令部分

#### ls 

|参数|作用|
|:-:|:-:|
|-a|显示隐藏文件|
|-l|显示文件详细性质|

#### mkdir

|参数|作用|
|:-:|:-:|
|-p|递归生成文件|

#### rm

|参数|作用|
|:-:|:-:|
|-r|递归删除文件|
|-f|强制删除文件|

#### cp
|参数|作用|
|:-:|:-:|
|-r|递归复制文件|

#### find
usage:$find [filename] "context"$
